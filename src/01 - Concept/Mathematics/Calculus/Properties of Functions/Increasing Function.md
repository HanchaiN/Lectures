# Increasing Function

$$
x_{1}<x_{2}\implies f(x_{1}) \le f(x_{2})
$$

Equiv. $f^{\prime}\left(x\right)\ge0$

For strictly-increasing function,

$$
x_{1}<x_{2}\implies f(x_{1}) < f(x_{2})
$$

Equiv. $f^{\prime}\left(x\right)>0$ (sufficient but not necessary)
