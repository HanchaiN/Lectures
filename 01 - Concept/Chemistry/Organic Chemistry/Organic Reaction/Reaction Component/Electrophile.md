# Electrophile

$\ce{E^{\oplus}}$ / $\ce{E}$ (can accept electron pair)

- Can react with negatively charged carbons or nucleophile
- $\ce{H^{\oplus}}$, $\ce{R^{\oplus}}$, $\ce{RC^{\oplus}=O}$, $\ce{O=N^{\oplus}=O}$
- $\ce{SO3}$, $\ce{BH3}$
