# Differentiability

- Differentiable at $a$: $f^{\prime}(a)$ is defined
- Differentiable on $(a,b)$: Differentiable for $x_0\in(a,b)$
- Imply [[Continuity|Continuity]] (the reverse is unnecessary true)
