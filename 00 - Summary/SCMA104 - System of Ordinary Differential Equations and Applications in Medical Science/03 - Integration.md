# Integration

- Integration
	- Definite: Riemann sum
	- Indefinite: Antiderivative $\int f^{\prime}(x) dx = f(x) + C$
		- First fundamental theorem of calculus: $\frac{d}{dx} \int f(x) dx = f(x)$
	- Second fundamental theorem of calculus: $\int_{a}^{b}f(x)dx = F(b)-F(a)$
